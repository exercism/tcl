# Reverse String

Reverse a string

For example:
input: "cool"
output: "looc"


## Submitting Incomplete Solutions
It's possible to submit an incomplete solution so you can see how others have completed the exercise.

## Running the tests
To run the test suite, execute one of the following commands:

```bash
tclsh reverse-string.test            # Will stop testing after the first failure.
RUN_ALL=1 tclsh reverse-string.test  # Will run all tests and report all failures.
```
